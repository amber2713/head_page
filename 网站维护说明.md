# 网站维护说明（如何修改作者页 & 如何新增作品）

本网站采用**单文件 `index.html` + `images` 文件夹**的极简结构。\
你只需要修改 `index.html` 和往 `images` 里放图片，就可以完成所有更新。

    your-repo/
    │
    ├── index.html
    └── images/
        ├── project1.jpg
        ├── project2.jpg

------------------------------------------------------------------------

## 一、如何新增一个作品

### 第一步：放入图片

把你的作品图片放进：

    images/

例如：

    images/my_new_work.jpg

------------------------------------------------------------------------

### 第二步：在 `index.html` 里找到这段

搜索：

``` html
<div class="projects">
```

你会看到已有的卡片代码。

------------------------------------------------------------------------

### 第三步：复制下面这一整段，粘贴到任意卡片下面

``` html
<div class="card">
  <img src="images/你的图片名.jpg" alt="项目图片">
  <div class="card-content">
    <h3>作品名称</h3>
    <p>这里填写作品的简介说明。</p>
    <a href="作品网址" target="_blank">访问作品 →</a>
  </div>
</div>
```

只需要改 3 个地方：

  位置                      改成什么
  ------------------------- ----------------
  `images/你的图片名.jpg`   你的图片文件名
  `作品名称`                作品标题
  `作品网址`                作品访问链接
  简介文字                  作品简介

完成！不需要改任何其他代码。

------------------------------------------------------------------------

## 二、如何修改作者介绍页

在 `index.html` 中搜索：

``` html
<section id="about"
```

你会看到作者介绍页面。

------------------------------------------------------------------------

### 修改小组简介

改这一段文字即可：

``` html
<p>
  本小组致力于将文学创作与数字技术相结合...
</p>
```

------------------------------------------------------------------------

### 新增 / 修改作者 Google Scholar

找到：

``` html
<ul class="authors">
```

每个作者是一行：

``` html
<li>作者 A：
  <a href="Google Scholar 链接" target="_blank">
    Google Scholar 页面
  </a>
</li>
```

想增加作者，**复制一行，多写一个即可**。

------------------------------------------------------------------------

## 三、常见错误

  问题         原因
  ------------ ----------------------------------------------
  图片不显示   图片没有放在 `images` 文件夹，或文件名不一致
  点击没反应   网址没写 `https://`
  页面没变化   GitHub Pages 需要等 10\~30 秒刷新

------------------------------------------------------------------------

## 四、更新网站流程（非常简单）

1.  改 `index.html`
2.  或往 `images` 里加图片
3.  Push 到 GitHub
4.  等待 GitHub Pages 自动更新

网站会自动刷新，无需任何部署操作。
